**Data Analytics Project: Maven Market**

**Overview**

This project provides a comprehensive analysis of General Store to identify key trends and actionable insights. By processing raw data into an interactive Power BI dashboard, the project enables stakeholders to make data-driven decisions regarding revenue optimization.

Dataset
Source: Maven market CSV file

Size: 8 CSV files.

**Description**: The dataset contains records of transactions, dates, customer demographics, and product details.

Tools
Power Query: Data extraction, cleaning, and transformation.

Power BI: Data modeling, DAX (Data Analysis Expressions), and visualization.

**Steps**

1. Data Loading & Cleaning (Power Query)
Extraction: Imported raw data from multiple CSV files.

Profiling: Performed Exploratory Data Analysis (EDA) to identify missing values, outliers, and inconsistencies.

Transformation: * Handled null values and corrected data types.

Removed duplicates and irrelevant columns.

2. Data Modeling
Established a Star Schema to optimize performance.

Created relationships between fact tables and dimension tables.

Developed DAX Measures for key performance indicators (KPIs) such as Month-over-Month (MoM) growth.

3. Dashboard Building
Designed a user-centric interface with consistent branding.

Used a mix of bar charts, line graphs, and heatmaps to visualize complex trends.
